# ☕ Cafetería Zarzamora - Digital Menu & Admin

> Proyecto de Menú Digital Dinámico para Cafetería Zarzamora.

Este repositorio contiene la solución digital completa para la Cafetería Zarzamora, incluyendo una landing page, un menú digital optimizado para dispositivos móviles y un panel de administración para gestión interna.

## 🚀 Despliegue en Vivo

Puedes ver el proyecto en vivo aquí:
- **Landing (Home):** [Link de Vercel](https://cafeteria-zarzamora.vercel.app/)
- **Menú Digital:** [Menu](https://cafeteria-zarzamora.vercel.app/menu.html)
- **Panel Admin:** [Admin Dashboard](https://cafeteria-zarzamora.vercel.app/admin.html)

## 📂 Estructura del Proyecto

```
cafeteria-zarzamora/
├── index.html          # Landing page principal
├── menu.html           # Menú interactivo para clientes
├── admin.html          # Panel de gestión administrativa
├── cafe_frames/        # Assets visuales y frames de animación
└── README.md           # Documentación del proyecto
```

## 🛠️ Tecnologías Utilizadas

- **HTML5 & Vanilla CSS3**: Estructura sólida y diseño premium sin dependencias pesadas.
- **JavaScript (Vanilla)**: Lógica de navegación y micro-animaciones fluidas.
- **Vercel**: Hosting de alta disponibilidad para carga instantánea en móviles.

## 📝 Instalación Local

```bash
# Clonar el repositorio
git clone https://github.com/TUGITHUBUSER/cafeteria-zarzamora.git

# Abrir con Live Server o directamente en el navegador
open index.html
```
